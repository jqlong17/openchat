# 02 执行计划：细化任务（滚动）

**定位**：在 [01plan.md](./01plan.md) 的**阶段目标与 DoD** 之下，记录 **更具体的** 拆解：按 PR、按模块、按责任人或按时间盒，便于多人协作与每日推进。

**TDD**：细化任务须体现 **红 → 绿 → 重构**；每个 PR 建议标注「**测试提交** / **实现提交**」或同一 PR 内先测后码，与 [01plan TDD 总则](./01plan.md#tdd-总则全阶段适用) 一致。

**规则**：

- **不删改** `01plan` 中的阶段编号与总 DoD；若总表变更，在 `01plan` 修订并在此文注明「对齐 01plan vx.x」。
- 本文可 **按阶段分节**，完成的小项打勾；过时任务可划掉或移到「已完成归档」小节。
- 仓库**第一次** `cargo init` / workspace 布局、OpenAPI 首检、健康检查等 **仍归属 01plan 阶段 0**，此处只写**细粒度步骤**（见下节示例）。
